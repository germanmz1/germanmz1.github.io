# Portfolio Personal – German Martinez

# Descripción
Este repositorio contiene el código fuente de mi portfolio personal,
desarrollado como un proyecto para presentar trabajos en Ciencia de Datos,
Desarrollo de Software y Ciencias Económicas.

# Objetivo del proyecto
Definir una estructura clara, mantenible y escalable para un portfolio
multidisciplinario, priorizando la semántica, la accesibilidad y la
separación de responsabilidades.

# Estructura del proyecto
HTML semántico para la estructura del contenido
CSS para la presentación visual
JavaScript para comportamiento dinámico (idioma, tema, navegación)

# Decisiones de diseño
Separación entre contenido estático y dinámico
Internacionalización controlada (ES / EN)
Navegación orientada a áreas de conocimiento
Proyectos presentados como estudios de caso
# Tecnologías utilizadas
HTML5
CSS3
JavaScript (ES6+)
